¿Que es HTML?

HyperText Markup Language(HTML), Es el lenguaje con el que se define el contenido de las páginas web.

Tim Berners-Lee

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, trabajador del CERN (Centro Europeo de Investigación Nuclear) propuso un nuevo sistema de "hipertexto" para compartir documentos.

¿Cómo surgió?

Tim Berners-Lee lo presentó a una convocatoria organizada para desarrollar un sistema de hipertexto para internet. Después de unir sus fuerzas con el ingeniero de sistemas Robert Cailliau, presentaron la propuesta ganadora llamada WorldWideWeb.

Etiquetas básicas










Son fragmentos de código que permiten crear elementos HTML, las más básicas son:
Etiqueta Función
body Para el contenido
head Para información sobre el documento
div División dentro del contenido
strong Para poner el texto en negrita
br Para saltos de línea
h1...h6 Para títulos en el contenido
img Para añadir imágenes al documento
ol-ul-li Listas
p Para parrafos
span Para estilos de una parte del texto
